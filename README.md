# Controle de Medição – Planejamento x Realizado v7.1 Corrigido

Sistema desenvolvido em **Python** para controle de medição, acompanhamento de **planejamento x realizado**, consolidação de dados e apoio à análise de produtividade e desempenho.

## Objetivo

Este sistema tem como finalidade organizar e consolidar informações de medição de forma estruturada, permitindo:

- acompanhamento de valores planejados e realizados;
- análise de produtividade;
- consolidação de indicadores;
- apoio à tomada de decisão;
- visualização organizada dos dados por meio da interface do sistema.
